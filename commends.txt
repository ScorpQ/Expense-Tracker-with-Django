- venv aktfileştirme
    source venv/Scripts/activate
    > aktifleşmiyorsa kaldır tekrar yükle
      rm -rf venv
      python -m venv venv

- run the project
    python manage.py runserver

- Başlamazsa local'de pg ayağa kaldırman lazım.
    docker-compose up

- bağımlılıkları kaydetmek için
    pip freeze > requirements.txt

- bağımlılıklar için
    pip install -r  requirements.txt

- migration for specific app 
    python manage.py makemigrations expenses
    python manage.py migrate

    - migration yaparken db'Den değil de kodun içini değiştirip migrate etmen best-practice

    - migration dosyalarını listelemek için 'python manage.py showmigrations'
